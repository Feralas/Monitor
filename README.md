This is the monitor package for Orchid CMS.

 * Board temperature
 * Network status
 * Uptime
 * CPU Load
 * Memory allocation
 * Available storage

## Installation

Pull in the package through Composer.

Run `composer require orchid/monitor` to get the latest stable version of the package.
